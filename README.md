# Supervisor

Supervisor install script.

## Requirements

* Ubuntu (tested on Ubuntu 22.04, Ubuntu 20.04, Ubuntu 18.04, Ubuntu 16.04, Ubuntu 14.04)

## Install

```
wget https://raw.githubusercontent.com/x-shell-codes/supervisor/master/supervisor.sh
sudo bash supervisor.sh
```

## Attentions

* When creating a swap area, there must be enough space for the file to be created.
* DO NOT RUN THIS SCRIPT ON YOUR PC OR MAC!
